Instructions to get started:

We will be using python 3.11.7

0. create .env file with the following variables:
    DB_USERNAME=
    DB_PASSWORD=
    DB_HOST=
    DB_NAME=
    STRIPE_PUBLISHABLE_KEY=
    STRIPE_SECRET_KEY=

1. install virtualenv
    pip3 install virtualenv

2. create virtualenv in the working directory:
    "python3 -m venv HHH"

2. Activate virtual env:
    Windows: "HHH\Scripts\activate"
    Mac: "source venv/bin/activate"

3. Install required dependencies:
    "pip3 install -r ./requirements.txt"

4. check environment dependencies:
    "pip list"

4. Run the server
    "python3 ./app.py" or "flask run"

5. To deactivate the venv:
    "deactivate"
